# GBP/USD Time Series Project

This project looks at the GBP/USD exchange rate using Prophet in R.

## What I did
- Downloaded daily exchange rate data from Yahoo Finance  
- Plotted the data to understand how it behaves  
- Used Prophet to build a forecasting model  
- Compared models with and without seasonality  

## What I found
- The data is quite volatile  
- There is no strong seasonal pattern  
- The trend changes over time  
- Forecasts have quite a lot of uncertainty
